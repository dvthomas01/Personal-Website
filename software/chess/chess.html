Chess Engine

  • Developed a chess engine using Python and the chess library, implementing the minimax algorithm with alpha-beta pruning for efficient move evaluation.
  • Integrated a graphical user interface using Tkinter, providing an interactive chessboard where users can play against the AI engine.
  • Implemented move validation and game state management to ensure legal moves and proper game progression.
  • Added depth-limited search to balance computational efficiency with playing strength, allowing for configurable AI difficulty levels.
