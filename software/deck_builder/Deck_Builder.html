Yu-Gi-Oh! Deck Builder

  • Built a Yu-Gi-Oh! deck builder that allows users to filter and search over 12,000 cards from the Yu-Gi-Oh! API, using criteria like archetype, level, type, name, and attributes.
  • Enabled users to manage their decks by saving favorite cards and removing unwanted ones for a customizable deck-building experience.
  • Integrated an AI-powered chatbot with the OpenAI API to provide users with news on competitive decks, trends, and Yu-Gi-Oh! tournaments.
  • Developed the backend using Node.js and Express.js, ensuring smooth server-side operations and API interactions.
  • Utilized EJS for dynamic templating and Bootstrap for a polished frontend design, with Axios handling efficient HTTP requests to the Yu-Gi-Oh! API.
