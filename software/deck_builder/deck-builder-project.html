Project Overview

Why: I built this to gain experience with public APIs and to ship an interactive product for Yu-Gi-Oh! players.

What: The app lets you search and filter over 12,000 cards, view details, and build a personal deck by saving/removing cards. It includes a deck advisor chat UI for meta/news guidance (backend integration planned).

How It Works

Backend: Express app with EJS view engine, static assets, and cookie-based favorites storage (Vercel-ready, serverless-compatible).

Frontend: EJS templates + Bootstrap + Vanilla JS; no bundling. State is managed with DOM and cookies.

Routes: Homepage, Favorites, Advisor; REST endpoints for favorites CRUD; POST search to YGOPRODeck.

YGOPRODeck API Usage

Base: https://db.ygoprodeck.com/api/v7/cardinfo.php

Filters map to UI controls: name, fName, attribute, archetype, type, level, race.

// Search route (POST /post-cards)
app.post('/post-cards', async (req, res) => {
  try {
    const { name, fName, attribute, archetype, type, level, race } = req.body;
    const params = new URLSearchParams();
    if (name) params.append('name', name);
    if (fName) params.append('fname', fName);
    if (attribute) params.append('attribute', attribute);
    if (archetype) params.append('archetype', archetype);
    if (type) params.append('type', type);
    if (level) params.append('level', level);
    if (race) params.append('race', race);

    const url = `https://db.ygoprodeck.com/api/v7/cardinfo.php?${params.toString()}`;
    const response = await fetch(url);
    const data = await response.json();
    res.render('search-post', { cards: data.data || [] });
  } catch (error) {
    res.render('index', { title: 'Search error' });
  }
});

Responses include card stats and image URLs suitable for grid display and detail overlays.

Advisor Chatbot

The advisor UI provides a chat interface for deck-building tips, meta trends, and tournament news. Backend AI integration is planned (OpenAI dependency present but not yet wired up). Guardrails will summarize sources and avoid prescriptive advice without context.
