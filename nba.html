NBA Chatbot

  • The "Cosine Similarity based Chatbot" project is a Flask application that employs cosine similarity to provide responses to user queries.
  • It utilizes Python along with the Natural Language Toolkit (nltk) and numpy/panda libraries for text processing and similarity calculations, as well as HTML for deployment.
  • The chatbot tokenizes user input, removes stopwords to reduce noise, and compares the input to a pre-defined list of questions stored in a CSV file.
  • When a match is found, the corresponding answer is returned to the user; otherwise, the chatbot responds with "We can't answer this".
