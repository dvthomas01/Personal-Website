Pac-Man Bot

  • Developed a Deep Convolutional Q-Network (DCQN) to play Pac-Man using PyTorch and OpenAI Gymnasium, achieving autonomous gameplay through reinforcement learning with average scores of 1000.
  • Implemented key components including epsilon-greedy exploration, experience replay buffer, and target network for stable Q-learning updates.
  • Designed a custom CNN architecture to process game frames, extracting relevant features for effective decision-making in the Pac-Man environment.
  • Utilized CUDA for GPU acceleration, optimizing training performance and enabling faster convergence of the Q-network.
